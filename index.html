14. (a) If $0 \leq a,$ then $|a|=a=-(-a)=|-a|,$ since $-a \leq 0.$ Similarly, if $a \leq 0,$
then $|-a|=-a=|a|,$ since $0 \leq -a. \ \ \square$
  (b) If $|a| \leq b,$ then naturally $0 \leq b.$ Now $|a| \leq b$ means that $a \leq b$ if $0 \leq a,$
moreover $a \leq b$ if $a \leq 0.$ Similarly, $|a| \leq b$ means $-a \leq b,$ and hence $-b \leq a,$
if $a \leq 0,$ and surely $-b \leq a$ if $0 \leq a.$ So, $-b \leq a \leq b.$ Conversely, if $-b \leq a \leq b.$ then $|a|=a \leq b$ if $0 \leq a,$ while $|a|=-a \leq b$ if $a \leq 0. \ \ \square$
  (c) From $-|a| \leq a \leq |a|$ and $-|b| \leq b \leq |b|$ it follows that $-(|a|+|b|) \leq a+b \leq |a|+|b| \ \Leftrightarrow \ |a+b| \leq |a|+|b|. \ \ \square$

15. If $x \neq y,$ then $$x^2+xy+y^2=\displaystyle\frac{x^3-y^3}{x-y}.$$ Problem 6(b) shows that the quotient on the right is always positive (since $0 <$ $x^3-y^3$ if $0 < x-y$ and $x^3-y^3 < 0$ if $x-y < 0$). Moreover, if $x=y \neq 0,$ then $0 < x^2+xy+y^2=3x^2.$

and $x^5-y^5 < 0$ if $x-y < 0$. In particular, if $x=y \neq 0,$ then $0 < x^4+x^3y+x^2y^2+xy^3+y^4=5x^4. \ \ \square$
