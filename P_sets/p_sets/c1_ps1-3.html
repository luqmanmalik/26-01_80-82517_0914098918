01. Prove the following:
(i) If $ax=a$ for some number $a \neq 0,$ then $x=1.$
(ii) $x^2-y^2 = (x-y)(x+y).$
(iii) If $x^2=y^2,$ then $x=y$ or $x=-y.$
(iv) $x^3-y^3 = (x-y)(x^2 + xy + y^2).$
(v) $x^n-y^n = (x-y)(x^{n-1} + x^{n-2}y + \dots + xy^{n-2} + y^{n-1}).$
02. What is wrong with the following "proof"? Let $x=y.$ Then $$ \require{cancel} \begin{equation} \begin{split} x^2 &= xy, \\ x^2-y^2 &= xy-y^2, \\ (x+y)(x-y) &= y(x-y), \\ x+y &= y, \ \ \ \ \color{grey}{\text{Err} \ \ \cancel{0 \vert a}} \\ 2y &= y, \\ 2 &= 1. \\ & \ \square \end{split} \end{equation} $$
03. Prove the following:
(i) $ \displaystyle \frac{a}{b} = \frac{ac}{bc}$, if $b,c \neq 0. $
(ii) $ \displaystyle \frac{a}{b} + \frac{c}{d} = \frac{ad+bc}{bd} $, if $b,d \neq 0.$
(iii) $(ab)^{-1}=a^{-1}b^{-1},$ if $a,b \neq 0.$ (To do this, you must
remember the defining property of $(ab)^{-1}$).
(iv) $ \displaystyle \frac{a}{b} \cdot \frac{c}{d} = \frac{ac}{db}, $ if $b,d \neq 0.$
(v) $ \displaystyle \frac{a}{b} \bigg/ \frac{c}{d} = \frac{ad}{bc}, $ if $b,c,d \neq 0.$
(vi) If $b,d \neq 0$, then $ \displaystyle \frac{a}{b} = \frac{c}{d} $ if and only if $ad=bc.$ Also determine
when $ \displaystyle \frac{a}{b} = \frac{b}{a}. $
